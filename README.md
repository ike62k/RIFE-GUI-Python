## 動作環境
- windows10 or windows11
- Python 3.12(以前のVerでも動く可能性はあります※未検証)
- (あったらいいな)Intel,AMD,NVIDIAの外付けGPU

## どんなソフトウェア？
映像の前後コマから間のコマを生成するソフトウェア[RIFE-ncnn-Vulkan](https://github.com/nihui/rife-ncnn-vulkan)をより便利に活用するためのソフトウェアです。<br>
